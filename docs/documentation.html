API Reference

WiBLE Class

  • begin(config): Initialize the library
  • startProvisioning(): Start BLE advertising
  • loop(): Handle events (call in loop)
  • isProvisioned(): Check status

Callbacks

  • onWiFiConnected(cb)
  • onProvisioningComplete(cb)
  • onError(cb)
  • onCustomDataReceived(cb)

Security Levels

Level Description
NONE No encryption (Development only)
SECURE ECDH Key Exchange + AES-256 (Recommended)
