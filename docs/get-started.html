Get Started

Follow these steps to integrate WiBLE into your ESP32 project.

1. Installation

PlatformIO (Recommended)

Add the following to your platformio.ini file:

lib_deps = 
    Chamath-Adithya/WiBLE@^2.0.0

Arduino IDE

1. Go to Sketch > Include Library > Manage Libraries

2. Search for WiBLE

3. Click Install

2. Partition Scheme (Critical!)

WiBLE uses Bluetooth and WiFi stacks, which requires a larger app partition.

PlatformIO Configuration

board_build.partitions = huge_app.csv
; OR if you need OTA:
; board_build.partitions = min_spiffs.csv

3. Hello World

Copy this code into your main.cpp or .ino file:

#include <WiBLE.h>

WiBLE provisioner;

void setup() {
    Serial.begin(115200);
    
    ProvisioningConfig config;
    config.deviceName = "My-ESP32";
    config.securityLevel = SecurityLevel::SECURE;
    
    provisioner.begin(config);
    provisioner.startProvisioning();
}

void loop() {
    provisioner.loop();
}
