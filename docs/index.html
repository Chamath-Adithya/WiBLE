5 Lines of Code.

WiBLE is designed to be a drop-in solution. Initialize, configure, and start. We handle the complex BLE/WiFi orchestration for you.

  • ‚úÖ No boilerplate
  • ‚úÖ Type-safe configuration

main.cpp 
#include <WiBLE.h>

WiBLE provisioner;

void setup() {
  ProvisioningConfig config;
  config.securityLevel = SecurityLevel::SECURE;
  
  provisioner.begin(config);
  provisioner.startProvisioning();
}

void loop() {
  provisioner.loop();
}
