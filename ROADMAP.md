# WiBLE Roadmap

This document outlines the planned features and improvements for WiBLE. Note that this roadmap is subject to change based on community feedback and priorities.

## v2.1 (Q2 2025)
- [ ] Bluetooth Mesh support
- [ ] Matter/Thread integration
- [ ] Flutter SDK for the companion app
- [ ] React Native SDK for the companion app

## v2.2 (Q3 2025)
- [ ] OTA firmware update support out-of-the-box
- [ ] Cloud integration examples (AWS IoT, Azure IoT)
- [ ] Provisioning analytics dashboard
- [ ] Multi-language support in documentation

## v3.0 (Q4 2025)
- [ ] BLE 5.3 feature support (e.g., periodic advertising with responses)
- [ ] Edge computing integrations
- [ ] AI-powered diagnostic and troubleshooting tools

We welcome community contributions to help accelerate this roadmap!
